# **App Name**: MindBloom

## Core Features:

- User Authentication: Secure login/registration for personalized experience.
- AI Expression Analysis: Use the device's camera and microphone to detect the user's stress levels based on facial and audio cues.
- Mental Health Screening: Administer PHQ-9 and GAD-7 tests.
- Personalized Resource Hub: Curate resources like videos and games.
- AI-Driven Chatbot: A conversational chatbot offers coping suggestions using a reasoning tool and refers users to professional help when needed.
- Anonymous Peer Support: Secure, moderated forum.
- Counselor Booking: Schedule appointments.
- Admin Analytics: Anonymous dashboards.
- Localization: Enable local customisations of the user interface to the local language

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to promote calmness and well-being.
- Background color: Light gray (#F5F5F5) to create a clean and soothing backdrop.
- Accent color: Gentle green (#90EE90) for positive and reassuring elements.
- Body and headline font: 'Alegreya', a serif font lending an elegant, intellectual, contemporary feel.
- Use minimalist icons to represent features and categories.
- Ensure a clean, intuitive layout with ample white space to reduce visual clutter.
- Incorporate subtle, calming animations for transitions and feedback.
